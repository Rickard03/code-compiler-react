# code-compiler-react




## Requirements?

1. Press "Clone or download" and download it as a .zip file. 
2. Extract the folder to your computer. 
3. Install NodeJS. It's required for it to work. https://nodejs.org/en/

## CLIENT SETUP

Type
```sh
cd client
```

Then simple type
```sh
npm install
```

Last step
```sh
npm run start
```



## SERVER SETUP


Open new terminal and type
```sh
cd server
```

Then type
```sh
npm install
```

Finally type
```sh
nodemon server.js
```
